# Opin — Просмотр оригинальных изображений Pinterest

[English](../README.md) · [한국어](README.ko.md) · [日本語](README.ja.md) · [简体中文](README.zh-CN.md) · [繁體中文](README.zh-TW.md) · [ไทย](README.th.md) · [Italiano](README.it.md) · **Русский**

![Opin — Просмотр оригинальных изображений Pinterest](../images/cover-en.png)

Opin — это расширение для браузера, которое позволяет открыть оригинальное изображение в высоком разрешении, стоящее за любым пином Pinterest. Оно добавляет кнопку рядом с кнопкой **Сохранить** в Pinterest — по клику оригинал в полном размере открывается в новой вкладке.

Оно создано для дизайнеров и исследователей, которые используют Pinterest для сбора референсов и нуждаются в исходных изображениях наивысшего качества.

## Возможности

- Добавляет кнопку **Просмотр оригинала** рядом с кнопкой «Сохранить» — как в сетке (ленте), так и на странице пина.
- Открывает изображение `/originals/` в полном разрешении в новой вкладке.
- Автоматически проверяет наличие оригинала и отключает кнопку, если его нет.
- Определяет видео-пины (у которых нет оригинального изображения) и помечает их соответствующим образом.
- Работает полностью в вашем браузере — **без сбора данных, без внешних серверов**.
- Многоязычный интерфейс: английский, корейский, японский, китайский упрощённый, китайский традиционный, тайский.

### Ручная установка (режим разработчика)

- **Chrome / Edge / Whale:** откройте `chrome://extensions`, включите **Режим разработчика** → **Загрузить распакованное расширение** → выберите папку `chrome`.
- **Firefox:** откройте `about:debugging#/runtime/this-firefox` → **Загрузить временное дополнение** → выберите `firefox/manifest.json`.

## Как пользоваться

1. Откройте Pinterest.
2. Наведите курсор на пин или откройте его страницу.
3. Нажмите кнопку Opin (красный значок **P** Pinterest) рядом с кнопкой **Сохранить**.
4. Изображение в оригинальном разрешении откроется в новой вкладке.

Opin не собирает и не хранит никаких персональных данных и никогда не связывается с внешними серверами. См. полную [Политику конфиденциальности](PRIVACY.ru.md).
